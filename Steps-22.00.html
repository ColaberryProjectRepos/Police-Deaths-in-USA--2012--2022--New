Adding Visual Chart:-Deaths by Weapon and Map 22

Visual 5. Deaths by Weapon(Word Chart)



  •  On the visualization pane click on more visualization and get Word to import it to your Visualization pane.
  •  Copy and paste the Time chart.
  •  change it to a word chart and fill up the fields as shown below. 
  • General: Properties, Size:
  •  Height=288
  •  Width=343
  •  Position:
  •   Horizontal=573
  •   Vertical=432
  •   Title=On, Text=Deaths by Weapon
  •   Effects=Visual border=On white 10% darker
  •   Keep everything else the same.
Visual
  •  Category=weapon
  •  Values = Total Deaths
  •  General
  •  Min num of words=1
  •  Max number of words=204
  •  Min font size =50%
  •  Max font size = 100%
  •  Special characters =on
  •   Stop words =on
  •   Rotate words = on

Visual 6. Map


  • Copy and paste Time chart and resize the visual to fit the space.
  • Change the chart type by selecting Map Chart from the visualizations pane 
  • Fill up figures provided below for the necessary fields
Build Visual Tab:
  • Location =State
  • Color Saturation=YOY%
  • Tooltip= Tooltips = Total Deaths, Death YOY%, Avg Tour Mths, ATM YOY%,      Killed by Gunfire, KbG%, KbG YOY%     
General: Properties, Size:
  •  Height=288
  •  Width=409
  •   Position:
  •  Horizontal=870
  •  Vertical=432
  • Title=On, Text=State,
  • Map settings, Style=USA:States, Show labels=Off
  •  Keep everything else the same.
  •  Color: 
  •  Format Style=Gradient
  •  Minimum= #AEBDCA
  •   Center= #2C5B85
  •   Maximum= #1362AA
  •   Values Min=0
  •   Center= 0.25
  •   Max= 0.5
  •   Category label=Off
