Steps
  • Copy and paste the Time chart and resize the visual to fit the space. Change the chart type by selecting Scatter Chart from the visualization pane. 
  •  Fill up the figures provided below for the necessary fields.
Build Visual Tab:
  • Values = Rank
  •  X-axis=Death per Month
  •  Y-axis =Death YoY%
  •  Size= Total Deaths
  •  Tooltips = Total Deaths, Death YOY%, Avg Tour Mths, ATM YOY%, Killed by Gunfire, KbG%, KbG YOY%
Format Visual Tab:
General: Properties, Size:
  •  Height=287
  •  Width=390
  •   Position:
  •   Horizontal=184
  •   Vertical=432
  •   Title=On, Text=Deaths by Rank, 
  •    Effects=Visual border=On, white 10% darker
  •   Keep everything else the same.
Visual: 
  •  X-axis: Title = On, Font =Segoe UI(9),Title Color= #4184C5       Y-axis:  Title = On, Title Color= #6A331B,Range,Min=Auto,Max=(Fx) Format Style= Field value and What field should we base this on=Death YOY%
  •  Legend=On, Match line color=On, 
  •  Title color= #A38600, Title=On, Text =KbG%
  •  Markers: Color Border=On, Size=10px
  •  Category label=On
  •   Color: 
  •  Conditional Formatting(fx), Color-Callout value, 
  •   Format Style=Gradient
  •   What field should we base this on?= killed by Gunfire%
