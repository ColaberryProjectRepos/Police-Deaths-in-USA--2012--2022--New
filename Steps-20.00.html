Visual 2. Line and Stacked Column (Age Group)



Steps
  •  Copy and paste the Time chart and resize the visual to fit the space.
  •   Fill up the figures provided below for the necessary fields
Build Visual Tab:
  •  X-axis = Age_Group, Sort in ascending order
  •  Column y-axis = Death per Month
  •  Line y-axis ==.Death YoY%, .ATM YoY%, .KbG%, .KbG YoY%,
  •  Tooltips = Total Deaths, Death YOY%, Avg Tour Mths, ATM YOY%, Killed by Gunfire,    KbG%, KbG YOY%
  •  After filling up the above rename fields, as per gif provided below
Format Visual Tab:
General: Properties, Size:
  • Height=344
  • Width=227
  • Position:
  • Horizontal=770
  • Vertical=88
  •  Title=On, Text= Age Group 
  • Keep everything else the same.
  • Font=DIN (17), Text color=#252423 , Background color #ACD0E7, Horizontal alignment=Center
  •  Effects: Background=On, Background color=#FFFFFF, Transparency=20%,
  •  Visual boarder=on, color=white 10% darker. 
Visual: 
  •  X-axis: Values: Font=Segoe UI(9), Font color=#000000
  •  Title = Off
  •  Y-axis: 
  •   Values: Font=Segoe UI Bold (9), Font color=#4184C5, Display Units=Auto, Title = Off
  •   Secondary y-axis=On
  •  Font=Segoe UI Bold (9), Font color= #6A331B, Display Units=Auto, Title = Off 
Legend=On, 
  •  Position=Top left, Style = Marker(circle default), Match line color=On, Title=Off, Text=Segoe UI(8),Color=#000000
  •  Gridlines: Horizontal=On, Style = Dotted, Vertical=Off
  •  Column: Color= #4184C5
  •  Lines: Colors: KbG YoY%= #EB895F, KbG%= #A38600, ATM YoY%= #9071CE, Death YOY%= #6A331B
  •  Shade area=On
  •  Markers= Off
  •  Data labels: Off
  •  Series labels: Off
