Download and Load CSV File to Power BI Desktop 8


 
Dataset Links: 
Download the dataset from Kaggle and save it on your desktop.
https://www.kaggle.com/datasets/mayureshkoli/police-deaths-in-usa-from-1791-to-2022
  1.       Download the dataset from Kaggle and save it on your desktop
  2.      Open Power BI Desktop
  3.       Click on the Get Data button.
  4.      Select  Text/CSV and click on Connect.
  5.       Select the ‘Police deaths in the USA from 1791 to 2022’ file and click on Open.
     A preview of the data will be displayed, click on Load/Transform
