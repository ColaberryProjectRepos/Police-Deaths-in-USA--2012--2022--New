Adding Year and State Slicers 16

1. Adding Year Slicer


  •   To Create the Year slicer, from the visualizations pane select Slicer
  •    Under the Fields Pane from the Calender table drag the year column to the fields value box
       Fill up the figures provided below for the necessary fields
General: Properties, Size
  •    Height=112
  •    Width=184 
  •    Position, 
  •    Horizontal=0 
  •    Vertical=89
  •    Title=On
  •    Text=Year
  •    Font=Din(17)
  •    Text color=#252423
  •    Background color=#ACD0E7
  •    Horizontal alignment=Center
  •    Effects, Background=On, 
  •    Background color=##FFFFFF,
  •   Transparency=20%
  •  Visual boarder=on, color= #E6E6E6(white 10% darker
Visual:
  •   Values, Font=Segoe UI (9), Font color=#070707
  •   Slicer=On

2. Adding State Slicer


  •    To Create the State slicer, from the visualizations pane select Slicer.
  •     Under the Fields Pane from the ‘Police deaths in USA from 2012 to 2022’ table drag the State  column to the fields value box
  •      Fill up the figures provided below for the necessary fields.
General: Properties, Size
  •   Height=112
  •   Width=184 
  •   Position, 
  •   Horizontal=0 
  •   Vertical=201
  •   Title=On
  •    Text=State
  •    Font=Din(17)
  •   Text color=#252423
  •   Background color= #ACD0E7
  •   Horizontal alignment=Center
  •    Effects, Background=On, 
  •    Background color=##FFFFFF,
